Question-1: What are the different ways to select an element in the DOM?

The DOM (Document Object Model) in JavaScript is a programming interface for web documents. It represents the page as a tree of objects, allowing JavaScript to dynamically access, manipulate, and update the content, structure, and style of a webpage.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML property allows you to get or set the HTML content inside an element, including any tags and formatting. On the other hand, innerText retrieves or modifies only the visible text within an element, ignoring any hidden elements and normalizing spacing. Lastly, textContent gets or sets all text within an element, including hidden elements, while preserving spacing but without interpreting HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a parent element handles events for its child elements using event bubbling. Instead of adding event listeners to multiple child elements, a single listener is added to a common ancestor to improve performance and simplify event handling.

Question-4: What is event bubbling in the DOM?

Event bubbling is a DOM mechanism where an event starts from the target element and propagates upwards through its ancestors. This allows parent elements to capture events triggered by their child elements, making event handling more efficient.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, use document.createElement('tagName'), then add it to the DOM using methods like parentElement.appendChild(childElement). To remove an element, use parentElement.removeChild(childElement) or childElement.remove().
